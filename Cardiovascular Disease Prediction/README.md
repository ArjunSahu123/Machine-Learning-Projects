# Cardiovascular Disease Prediction

## Project Overview

This project focuses on predicting the likelihood of cardiovascular disease using patient health and lifestyle data. The objective is to build a machine learning model that can identify individuals who may be at higher risk based on various medical and behavioral factors.

By analyzing patterns in the dataset, the model helps in understanding how different features contribute to heart-related conditions. Such predictions can support early diagnosis and allow timely preventive measures.

---

## Dataset Description

The dataset contains multiple features that capture a patient’s overall health condition, lifestyle habits, and medical history. Each feature provides useful information that can be used to assess cardiovascular risk.

| Feature                     | Description                                      |

| General Health              | Overall health status of the individual          |

---

## Key Insights

Exploratory data analysis revealed several meaningful patterns:

* Age is a strong indicator, with older individuals showing a higher risk of cardiovascular disease.
* Individuals with higher BMI tend to have an increased probability of developing heart-related conditions.
* Regular physical activity generally reduces risk, although its impact may vary across age groups.
* Dietary habits play an important role. Higher intake of fruits and green vegetables is associated with lower risk, while frequent consumption of fried foods shows the opposite trend.
* Smoking is a significant contributing factor to cardiovascular disease.
* Some medical conditions such as cancer, arthritis, diabetes, and depression did not show a strong direct relationship with heart disease in this analysis.

## Conclusion

This project highlights how machine learning can be applied to healthcare data to identify risk factors associated with cardiovascular disease. By leveraging these insights, it becomes possible to support early detection and improve preventive care strategies.

Overall, the model and analysis demonstrate the value of data-driven approaches in understanding and managing cardiovascular health.
